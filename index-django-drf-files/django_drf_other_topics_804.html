804. Co to jest Web Storage API?

    Web Storage API
    to zestaw narzędzi, który oferują nam nowoczesne przeglądarki internetowe. Dostarcza mechanizmy, dzięki którym
    możemy przechowywać dane lokalnie przy pomocy pary klucz-wartość, w prosty sposób i w większej ilości niż za pomocą ciasteczek.
    Jest szczególnie przydatny w aplikacjach, które wymagają personalizacji, cache'owania danych lub działania offline.
    Jednak należy pamiętać o potencjalnych zagrożeniach, takich jak podatność na ataki XSS, i stosować odpowiednie środki bezpieczeństwa.

Web Storage API obejmuje dwie główne technologie:
    Local Storage – dane są przechowywane na stałe (do momentu ich usunięcia ręcznie przez aplikację lub użytkownika).
    Session Storage – dane są przechowywane tylko na czas trwania sesji przeglądarki (są usuwane po zamknięciu karty lub przeglądarki).
