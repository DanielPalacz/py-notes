Przeglądarka wysyła HTTP request. Serwer zwraca HTML.
I dalej przeglądarka (uproszczona perspektywa):
  analiza/parsing HTML → tworzenie DOM
  pobiera CSS → tworzy CSSOM
  łączy DOM + CSSOM → tworzy Render Tree
  wykonuje layout (reflow) i paint
