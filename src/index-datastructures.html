LIFO (stack) and FIFO (queue):

Several options to implement stack and queue in Python.

Why 'deque' structure from collection module is good choice?
How to implement LIFO (stack) by deque?
    from collections import deque

    stack = deque()

    # Operations with stack
    stack.append(1)  # Push 1
    stack.append(2)  # Push 2
    stack.append(3)  # Push 3
    print(stack)  # deque([1, 2, 3])

    top = stack.pop()  # Pop (returns 3)
    print(top)  # 3
    print(stack)  # deque([1, 2])
How to implement FIFO (queue) by deque?
    from collections import deque

    queue = deque()

    # Operacje na kolejce
    queue.append(1)  # Enqueue 1
    queue.append(2)  # Enqueue 2
    queue.append(3)  # Enqueue 3
    print(queue)  # deque([1, 2, 3])

    front = queue.popleft()  # Dequeue (zwraca 1)
    print(front)  # 1
    print(queue)  # deque([2, 3])
