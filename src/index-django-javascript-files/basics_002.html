002. Jak osadzać kod JavaScript na stronie (wykorzystując tag script)?

Jakie są możliwości? Kiedy skrypt jest 'blokujący' i co to oznacza?

    1. Wstawianie JavaScriptu w pliku HTML (inline)
        a) W sekcji <head> (rzadko stosowane)
        b) W sekcji <body> (zalecane na końcu)

    2. Korzystanie z zewnętrznego pliku JavaScript (tag script)

    3. Atrybuty asynchroniczne i opóźnienie wczytywania
        a) defer (skrypt jest wykonywany po zakończeniu analizy całego HTML)
            - Używaj defer dla skryptów, które muszą manipulować DOM (np. frameworki frontendowe lub logika interfejsu użytkownika
        b) async (skrypt jest wykonywany równolegle z ładowaniem reszty dokumentu HTML)
            - Używaj async dla skryptów niezależnych, takich jak analityka lub reklamy.
        c) brak async i defer: skrypt jest blokujący, wczytanie/wykonanie skryptu zatrzymuje analizowanie HTML do czasu jego zakończenia

    4. Osadzanie kodu JS w atrybutach HTML (niezalecane)
